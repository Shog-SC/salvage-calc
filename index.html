🛠️ Salvage Calc 4.4

Convertir ton Construction Salvage (SCU) en Construction Materials (CMAT) + valeur estimée avec les prix UEX.

Étape 1

Entrer ton Salvage

Ce que sort ton Reclaimer / Vulture : des caisses de Construction Salvage (ex : 15 caisses de 16 SCU = 240 SCU).

Le total sera recalculé automatiquement.

Rendement vers Construction Materials

Ratio estimé après conversion en Construction Materials. Tu peux ajuster selon tes propres tests.

Exemple : 20% → 100 SCU de Salvage → 20 SCU de CMAT.

Étape 2

Prix UEX – Construction Materials (CMAT)

L’outil va récupérer les prix de vente UEX pour Construction Materials (CMAT) et calculer la valeur de ta cargaison.

Données prix via API UEX : uexcorp.space – endpoint /2.0/commodities_prices?commodity_name=construction-materials.

Note

➜ Les prix UEX sont communautaires, donc approximatifs.
➜ Le rendement (%) est configurable pour que tu puisses l’ajuster selon tes propres runs Reclaimer/Vulture.
➜ Aucun compte ni clé API n’est nécessaire pour cette version.
